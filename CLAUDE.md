# 打ち合わせ日時コピペツール

## 概要
打ち合わせの候補日時をカレンダーから選択し、テキストとしてクリップボードにコピーできるWebツール。
参考サイト: https://pa4fic.com/meeting/

## 技術構成
- 単一HTMLファイル (`index.html`) にCSS・JSをすべてインライン記述
- 外部ライブラリ・フレームワーク不使用（バニラJS）
- フォント: Google Fonts `Noto Sans JP`（400, 700）

## ファイル構成
```
appointment/
├── index.html       # アプリ本体（HTML/CSS/JS一体）
└── CLAUDE.md        # 本ファイル
```

## 画面構成

### 1. ヘッダー (.header)
- タイトル「打ち合わせ日時コピペツール」
- 表示時間帯の選択: 開始時刻・終了時刻のドロップダウン（0〜23時、デフォルト9〜18時）
- 時間単位の切り替えボタン: 60分 / 30分

### 2. 週ナビゲーション (.week-nav)
- 前週 (◀) / 次週 (▶) ボタン
- 現在表示中の週範囲ラベル（例: `3/30 〜 4/5`）

### 3. カレンダーグリッド (.cal-wrap > .cal-grid)
- CSSグリッド: 時間ラベル列(44px) + 7日分の列
- 日付ヘッダー (.dh): 曜日・日付・祝日名を表示
- タイムスロット (.slot): タップで候補を選択

### 4. 結果パネル (.result-wrap)
- 選択した候補一覧の表示
- 「クリア」ボタン: 全候補を削除
- 「まとめてコピー」ボタン: 全候補をクリップボードにコピー

### 5. トースト通知 (.toast)
- コピー成功時などに画面下部に一時表示（2秒後に自動消滅）

## 主要機能の仕様

### カレンダー表示
- 月曜始まりの週単位で7日間を表示
- 金・土曜日にアクセスした場合、翌週をデフォルト表示
- 過去の時間帯はグレーアウトして選択不可 (.past)

### 時間スロット選択（2タップ方式）
1. **1タップ目**: 開始時刻を仮選択 → pendingStart に保存、セルをハイライト (.pending-start)
2. **2タップ目（同じ日）**: 終了時刻を確定 → ranges 配列に追加
   - 終了 < 開始の場合は自動で入れ替え
   - 既存の重複範囲は自動除去
3. **別の日をタップ**: pending がその日に移動

### 選択範囲の表示
| CSSクラス | 役割 |
|-----------|------|
| .range-single | 1スロットのみの選択（開始=終了） |
| .range-start | 範囲の開始スロット |
| .range-mid | 範囲の中間スロット |
| .range-end | 範囲の終了スロット |

### コピー出力フォーマット
```
・4/2(木) 13:00〜15:00
・4/3(金) 10:00〜12:00
```
- 各行は「・」で始まる
- 日付フォーマット: `M/D(曜) HH:MM〜HH:MM`

## 曜日・祝日の色分け
- **日曜**: 赤テキスト + 薄赤背景 (.dh-sun)
- **土曜**: ティールテキスト + 薄ティール背景 (.dh-sat)
- **祝日**: 赤テキスト + 薄赤背景 (.dh-holiday-bg) + 祝日名表示

## 祝日データ
- `HOLIDAYS` オブジェクトにハードコード（2024〜2027年）
- キー形式: `"YYYY/M/D"`（ゼロ埋めなし）
- 値: 祝日名（例: `"元日"`, `"振替休日"`）

## レスポンシブ対応
- ブレークポイント: 768px
- モバイル時: パディング縮小、スロット高さ36px、グリッド最小幅360px
- `touch-action: manipulation` でモバイルタッチ最適化

## JS主要変数
| 変数 | 型 | 説明 |
|------|----|------|
| ranges | Array<{dk, startV, endV}> | 確定済みの時間範囲リスト |
| pendingStart | {dk, v} \| null | 1タップ目の仮保存 |
| startHour | number | 表示開始時刻（デフォルト: 9） |
| endHour | number | 表示終了時刻（デフォルト: 18） |
| unitMin | number | 時間単位 60 or 30 |
| weekOffset | number | 現在週からのオフセット |

## JS主要関数
| 関数 | 説明 |
|------|------|
| init() | 初期化、セレクトボックス生成、イベント設定 |
| buildCalendar() | カレンダーグリッドのDOM生成 |
| onSlotClick(dk, v) | スロットタップ時の選択ロジック |
| getRangeRole(dk, v) | スロットが選択範囲内かどうか判定 |
| formatRange(r) | 範囲オブジェクトを表示用テキストに変換 |
| updateResult() | 結果パネルの表示を更新 |
| getResultText() | コピー用テキストを生成 |
| copyResult() | 結果をクリップボードにコピー |
| clearAll() | 全選択をクリア |
| showToast(msg) | トースト通知を表示 |
| isPast(d, v) | 過去の時間帯かどうか判定 |
| slotLabel(v) | 数値を "HH:MM" 形式に変換 |
| getWeekDays(offset) | 指定オフセットの月〜日の Date 配列を返す |
| dateKey(d) | Date を "YYYY/M/D" 形式の文字列に変換 |
